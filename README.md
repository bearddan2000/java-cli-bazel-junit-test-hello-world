# java-cli-bazel-junit-test-hello-world

## Description
A POC for bazel app using JUnit.
Writes test results to console.

## Tech stack
- java
- bazel
  - junit

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
